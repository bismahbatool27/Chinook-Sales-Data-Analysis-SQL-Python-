# 🎵 Chinook Database Analysis (SQL Server + Python)

## 📌 Project Overview

This project demonstrates an end-to-end data analysis workflow using the Chinook music store dataset.

The objective of this project was to:

- Convert raw CSV files into structured SQL tables
- Design a relational database with Primary and Foreign Key constraints
- Perform business-focused SQL analysis
- Visualize insights using Python

This project highlights how structured data can be transformed into meaningful business insights.

---

## 🛠 Tools & Technologies Used

- SQL Server
- Python
- Pandas
- Matplotlib
- SQLAlchemy
- PyODBC
- Jupyter Notebook

---

## 🗄 Database Design

- Created a new database in SQL Server
- Defined Primary Keys for all tables
- Established Foreign Key relationships between tables

The dataset was successfully converted from CSV format into a fully structured relational database.

---

## 📊 Key Analysis Performed

- Top Revenue-Generating Tracks
- Revenue by Country
- Top Performing Artists
- Revenue by Genre
- Playlist Popularity Analysis
- Monthly Album Sales Performance

---

## 📈 Key Findings

- Rock is the highest revenue-generating genre ($826.65).
- The USA generates the highest total revenue ($523.06).
- Iron Maiden, U2, and Metallica are among the top revenue-driving artists.
- Revenue is concentrated among a limited number of genres.
